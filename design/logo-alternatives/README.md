# Logo alternatives

Three candidate replacements for the current `public/favicon.svg` (the gold
Scrabble tile "S₁"). Each is a drop-in replacement: same 32×32 viewBox, same
site palette (forest green `#162013`, sage `#e8efe6`, gold `#e8c860`), same
Georgia serif as the tile.

Open `preview.html` in a browser to compare them side by side at full size,
favicon size (16px, on dark and light tab backgrounds), and the mobile-bar
size (20px).

1. **`1-headword.svg` — The Headword.** The initials set like the site's
   dictionary headwords: serif bold gold `S·T` with the syllable interpunct
   from *Sam·my Taub·man*, plus a tiny italic *n.* where the tile's point
   value used to sit.
2. **`2-pronunciation.svg` — The Pronunciation.** The initials as a
   dictionary pronunciation guide: italic sage `st` between gold slashes,
   `/st/`.
3. **`3-tile-rack.svg` — The Rack.** Keeps the Scrabble tile but personal:
   both initials played side by side like tiles on a rack, each with its
   real point value (S=1, T=1), slightly askew as if hand-placed.

To adopt one, copy it over `public/favicon.svg` and delete this folder.
